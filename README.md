# PassOP - Password Manager

A secure password management application that allows users to store, manage, and access their website credentials in one place.

## Features

- Add new passwords
- Save website URLs and usernames
- View stored credentials
- Edit existing passwords
- Delete passwords
- User-friendly interface
- Full-stack architecture

## Tech Stack

### Frontend
- React
- Vite
- JavaScript
- CSS

### Backend
- Node.js
- Express.js

### Database
- MongoDB
